다양한 방법으로 실시간 스트리밍을 시청하세요

웹 브라우저에서 시청

스트림 키를 입력하고 시청하기 버튼을 클릭하세요

HLS 지원 브라우저에서 바로 시청 가능합니다

추천: Chrome, Safari, Edge 브라우저에서 최적의 시청 경험을 제공합니다.
오버레이 기능: 실시간 시간정보(우측하단)와 서비스명(좌측상단)이 자동 표시됩니다. 서비스명 변경은 위의 버튼을 클릭하세요.

시청 방법

웹 브라우저

가장 간단한 방법입니다. 위의 플레이어에서 바로 시청하세요.

지원: Chrome, Safari, Edge
화질: 자동 조절
VLC 플레이어

고화질 시청을 원하시면 VLC를 사용하세요.

URL 형식: FLV 스트림
지연시간: 최소
OBS Studio

방송인이라면 OBS의 소스로 추가해서 시청할 수 있습니다.

소스 타입: 미디어 소스
용도: 리액션 방송
VLC로 스트림 시청하기
1VLC 플레이어 실행

VLC Media Player를 실행합니다. 없다면 여기서 다운로드

2네트워크 스트림 열기

미디어 → 네트워크 스트림 열기 (Ctrl+N)

3스트림 URL 입력

다음 형식으로 URL을 입력하세요:

http://ai.gzonesoft.com:18001/live/[스트림키].flv
4재생

재생 버튼을 클릭하면 스트림이 시작됩니다.

OBS로 스트림 가져오기
1소스 추가

OBS의 소스 패널에서 + 버튼 → 미디어 소스 선택

2소스 이름 설정

적당한 이름을 입력하고 확인

3스트림 URL 설정

로컬 파일 체크 해제 후, 다음 URL 입력:

http://ai.gzonesoft.com:18001/live/[스트림키].flv
4완료

확인을 누르면 스트림이 OBS 소스로 추가됩니다.
